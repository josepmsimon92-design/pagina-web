# Pagina WEB

Proyecto de página web personal.

## Estructura

```
Pagina WEB/
└── README.md
```

## Cómo empezar

Clona el repositorio y abre `index.html` en tu navegador.

```bash
git clone https://github.com/josepmsimon92-design/pagina-web.git
cd pagina-web
```

## Contribuir

1. Haz un fork del repositorio
2. Crea una rama: `git checkout -b mi-mejora`
3. Haz tus cambios y un commit: `git commit -m "Descripción del cambio"`
4. Abre un Pull Request
